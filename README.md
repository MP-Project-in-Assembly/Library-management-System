In this Project we create "Library Management System".

The System consist of 5 main functions:
  1 - Add new member to the system (create new account).
  2 - Sign in the System if the member already exists.

For Each member:
  3 - Add Book.
  4 - Search for a Book.
  5 - Delete Book.
  6 - View members
  7 - Display all books.
  8 - Exit a programm.
